Zomato Delivery Delay Analysis | SQL + Python + Power BI
📌 Project Overview
This project analyzes delivery delays in food delivery operations using Zomato dataset. The objective is to identify key factors causing delays and provide actionable insights to improve delivery efficiency and customer satisfaction.

🎯 Business Problem
Delayed deliveries directly impact customer experience and retention. This project aims to:

- Identify patterns in delivery delays  
- Analyze impact of time, location, and order volume  
- Detect operational inefficiencies  
- Support data-driven decision-making for logistics optimization  

🛠️ Tools & Technologies
- SQL – Data cleaning and transformation  
- Python (Pandas, Matplotlib/Seaborn) – Exploratory Data Analysis  
- Power BI – Dashboard and visualization  
- Excel/CSV – Dataset
